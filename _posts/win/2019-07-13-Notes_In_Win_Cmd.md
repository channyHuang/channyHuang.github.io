# Notes_In_Win_Cmd

## 修改默认编码

按组合快捷键“win+ R”打开运行，输入cmd，打开命令提示符

输入   chcp

显示默认编码 936，即“gbk”编码

输入   chcp 65001

表示将编码转换为“utf-8”
